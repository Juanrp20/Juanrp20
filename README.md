## 📂 Proyectos Principales

Aquí encontrarás algunos de los proyectos en los que he trabajado durante mi aprendizaje y práctica como desarrollador.

🔹 **AirPOOrt: Sistema Integral de Gestión Aeroportuaria con Múltiples Roles**
Sistema desarrollado en Java con Programación Orientada a Objetos para gestionar las operaciones principales de un aeropuerto.
Permite administrar aviones, vuelos, pasajeros, trabajadores, equipajes y reservas, utilizando clases, herencia y encapsulamiento para modelar los diferentes componentes del sistema.
Incluye funcionalidades como registro de vuelos, gestión de reservas, mantenimiento de aviones y persistencia de datos mediante serialización.

🔹 **FREE-TOUR: Diseño y Modelamiento de Base de Datos para una Plataforma de Turismo Colaborativo con Guías Locales**
Proyecto enfocado en el diseño conceptual, lógico y físico de una base de datos para una plataforma que conecta turistas con guías locales que ofrecen recorridos turísticos.
Se definieron historias de usuario, modelo entidad-relación, normalización y estructura de tablas en Microsoft SQL Server, permitiendo gestionar usuarios, tours, reservas, reseñas y contenido multimedia dentro del sistema.

🔹 **Análisis Comparativo del Consumo de Memoria en Algoritmos de Ordenamiento: MergeSort, QuickSort, CountingSort y RadixSort para Grandes Volúmenes de Datos**
Estudio comparativo desarrollado en Java para analizar el consumo de memoria y rendimiento de diferentes algoritmos de ordenamiento:
MergeSort, QuickSort, CountingSort y RadixSort.

Se realizaron múltiples ejecuciones con arreglos de distintos tamaños para evaluar su escalabilidad y eficiencia, concluyendo que QuickSort presentó el menor consumo de memoria en promedio.
